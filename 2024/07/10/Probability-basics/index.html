機率是什麼，簡介集合論及基礎機率名詞

Posted on 2024-07-10 Edited on 2024-07-14 In Computer Science , Probability

簡介現代機率論的基礎與基本概念。

免責聲明

以下是我個人學完一年的機率，自己對機率論的理解。
單純是我在個人部落格上的心情抒發及學習筆記，不是什麼嚴謹的數學。
因此請不要完全相信，更不要作為文獻參考。
有什麼錯誤的地方還請不吝賜教 !

前言 什麼是機率

機率其實是人類為某個 “有可能發生之事件” 的發生給出的 “信心之度量”。例如天氣預報說明天有 50% 的機率降雨，代表我們對 “明天下雨” 這件事的發生，約有一半的信心。

但是這樣看不見、摸不著的 “信心” 之 “度量”，每個人都可以有自己的一套說法、標準。我們又如何能夠了解彼此口中的信心究竟為何 ? 又如何說服對方相信自己的一套說法呢 ?

例如: 明天我小考拿到滿分的機率有 0.87 、記者說法院駁回上訴的機率是 200%
我們無法確認彼此口中對信心給的度量之意義究竟為何。

因此現代的機率論體系誕生了。現代機率論是以集合論為基礎，再加上 Kolmogorov 提出的機率 3 公理誕生的世界觀。
只要你認同 Kolmogorov 提出的機率三公理，也就是這種度量應該遵守的特性。你就可以相信別人依循這個體系所得出來的機率值，也就是對某事件的信心之度量。
有了這樣的度量體系，你才可以了解別人口中對某事件發生之信心具體的程度為何。

而我們的機率論就是在討論，依循這樣的體系，我們的機率到底又是什麼 ? 有什麼樣的特性 ?

Fun: 之前在某 Facebook 社團中，看到有人在討論某道機率的題目時說: 這不是很直觀嗎 ? 下面留言馬上就有人反駁道: 機率從來就不是直觀的東西。
這真是說得太好了，機率從來就不是什麼直觀的東西。我們如何能根據機率論，給出我們對某事件發生信心的度量，是有嚴格的邏輯在背後的。因此也請大家對每道機率問題抱持嚴謹的態度 !

簡介集合論

What is 集合

A set is a collection of objects without order.
集合 是一堆彼此不相同的東西擺在一起表示的東西。注意這些東西彼此不重複，且沒有先後關係。

舉例而言，像是 A={1, 2, 3}、A={e | e=1 or e=2 or e=3}、A={x | x is natural number}，這裡的 A 就是一個集合。
其中 | 類似英文的 such that 的意思。

Each object is called an element of the set.
而這些擺在一起的東西，可以被稱作是這個集合的元素。

集合也可以是集合中的元素歐，例如 A={ {1, 2, 3}, {4, 5}, {1} } 也是一個合法的集合。

沒有元素的集合稱為空集合，記做 A={}=$\emptyset$。

集合的一些符號

  1. Membership ($\in$)

    $e \in A$ means that $e$ is an element of set $A$.

  2. Inclusion ($\subset$)

    A set $B$ is a subset of another set $A$ $\iff$ $\forall e \in B$, $e \in A$ as well.

  3. Equality ($=$)

    $A=B \iff A\subset B$ and $B\subset A$.

  4. Unerversal ($S$)

    $S$ is a set of all the elements we interested in.

集合的一些運算

  1. Complement ($A^c$)

    $x \in A^c \iff x \notin A$.

  2. Intersection ($\cap$)

    $x \in A \cap B \iff x \in A$ and $x \in B$.

  3. Union ($\cup$)

    $x \in A \cup B \iff x \in A$ or $x \in B$.

  4. Difference ($-$)

    $x \in A-B \iff x \in A$ and $x \notin B$.

    Note: $A-B=A\cap B^c$.

  5. Product ($\times$)

    $A \times B =${$(a,b) | a \in A$ and $b \in B$}. Where (a,b) is an ordered tuple.

  6. Power set ($2^A$)

    $2^A$ is the set of all posible subsets of set A.

    i.e. $2^A =${$e | e \subset A$}.

集合的一些定理

  1. Distribution Law

    1. $A \cup (B \cap C)=(A \cup B) \cap (A \cup C)$
    2. $A \cap (B \cup C)=(A \cap B) \cup (A \cap C)$

  2. De Morgan's Law

    1. $(A \cup B)^c = A^c \cap B^c$
    2. $(A \cap B)^c = A^c \cup B^c$

    Note: De Morgan's Law 串聯了三個基礎的集合操作，補集、聯集、交集。

兩個常用的形容詞

  1. Mutually Exclusive (ME)

    A collection of sets $A_1, A_2,…,A_n$ is mutually exclusive. $\iff$ $A_i \cap A_j = \emptyset$, $\forall i \neq j$.

  2. Collectively Exhaustive (CE)

    A collection of sets $A_1, A_2,…,A_n$ is collectively exhaustive. $\iff$ $\bigcup^{n}_{i=1} A_i=S$.

Partition 分割

A partition of set $A$ is a collection of “nonempty subsets” of set $A$ such that all the elements of $A$ appear exactly one of these subsets.

i.e. A collection of nonempty subsets of set $A$ such that these subsets are ME and CE.

e.g. A={x, y, z}, a partition of A is { {x}, {y, z} }.

簡單機率名詞

Random experiments (隨機試驗), Trial

在相同條件下，一個結果不能被確定的試驗，就稱為 random experiments。

每次重複這個 random experiments, 我們就稱做了一個 trial。

不管是進行的方式不同，或是觀察的東西不同，都為不同的 random experiments。

e.g. 投擲一個硬幣，觀察其是否為正面朝上 即為一 random experiments。

Outcome, Event, Sample Space

outcome 是一個 random experiments 中，可能觀察到的結果。

event 是一些 outcome 的集合。

sample space 是所有 outcome 的集合。更精準地說，這些 outcome 要滿足 finest-grain (一個 outcome 被觀察到，就不能有其他 outcome 被觀察到)、ME、CE。

e.g. 這一通電話的到來到下一通電話的到來間所需要等待的時間。 4s 是一個 outcome， 4~6s 間 是一個 event， {$t \in \mathbb{R} | t > 0$} 是 sample space。
